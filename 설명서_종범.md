<img src="./4_분석/분석_BI분석_화면상세.png">

- 분석 데이터를 기반으로 행렬 기준으로 통계화 한 자료를 조회하는 화면

- 통계자료를 위젯화하여 보기 위한 기능
- 위젯은 최대 5개 까지 설정하여 한 화면에 볼 수 있음

|버튼|상세|
|---|---|

|<img src="./4_분석/분석_BI분석_위젯설정_선그래프.png">|선택한 항목의 통계자료를 선그래프로 조회하는 위젯|

|<img src="./4_분석/분석_BI분석_위젯설정_표그래프.png">|전체 통계 자료를 조회하는 위젯|
|<img src="./4_분석/분석_BI분석_위젯설정_필터 그래프.png">|전체 통계자료를 필터 조건을 추가하여 조회하는 위젯|

- 또한 항목별로 행렬화 한 후 이를 기준으로 통계자료를 산출하여 볼 수 있음.

|버튼|상세|
|---|---|

|<img src="./4_분석/분석_BI분석_위젯설정상세_레이아웃.png">|다수의 결과 위젯 레이아웃 변경 |

  - Column : 열
  - Row : 행
  - Measure : 통계 방법(개수, 합계, 평균)
